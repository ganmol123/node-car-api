# node-car-api 
Whitepanda Backend Dev Task

A Car Rental Agency api to perform basic operations like booking a car, adding cars, deleting a car, etc.

Features Implemeted:
- Get all car list
- Add a new Cars
- Fetch all bookings
- Make a booking
- Delete a car (Only if not previously booked)
